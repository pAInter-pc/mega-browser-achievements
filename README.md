# Mega Browser с ачивками

Небольшой экспериментальный веб-браузер на Python + PyQt6 с системой достижений, тёмной темой и интерфейсом в стиле Chrome (я бы поспорил).

## Возможности

- Вкладки (обычные и помеченный “инкогнито”-режим).
- История, закладки, список загрузок.
- Тёмная тема, стилизованная под Chrome.
- Система ачивок:
  - ачивки за первые действия (первый запуск, ввод URL, домашняя страница);
  - прогресс за вкладки, историю, закладки, загрузки;
  - “поведенческие” ачивки (назад/вперёд, обновления, ночной запуск, меню);
  - более сложные ачивки: марафон, ежедневный запуск, wiki/видео/шоп-сайты, 404, плохое соединение и т.д.
- Всплывающий оверлей “как в Steam” при получении ачивки.
- Подробный список ачивок в виде карточек (как в Doors: Roblox).

## Установка из исходников

Требования:

- Python 3.14 (или совместимая версия, в которой собран PyQt6).
- Windows (проверялось на Windows 10/11).

Установите зависимости:

```bash
python -m pip install -r requirements.txt
```

Запуск браузера:

```bash
python browser.py
```

## Сборка .exe (Windows)

Сначала установите PyInstaller:

```bash
python -m pip install pyinstaller
```

Затем из папки проекта:

```bash
pyinstaller --noconfirm --windowed --name "MegaBrowser" browser.py
```

Исполняемый файл будет лежать в `dist/MegaBrowser/`.

## Ачивки

Примеры ачивок:

- **Привет, мир!** — первый запуск браузера.  
- **Первый шаг** — вручную введённый URL.  
- **Многозадачник** — 5 открытых вкладок.  
- **Память как у слона** — длинная история.  
- **Лутер** — первая загрузка файла.  
- **Ночной наблюдатель** — запуск после полуночи.  
- **Марафонец** — сессия 2+ часа.  
- **Энциклопедист** — несколько wiki-доменов.  
- **404 Hunter** — несколько страниц 404.  
- и многие другие.

Полный список и статус получения видно в меню:

- “Список ачивок (списком)”  
- “Список ачивок (карточки)”.
